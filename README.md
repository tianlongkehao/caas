# 代码使用说明
- 1.获取代码
- 2.构建项目：mvn eclipse:eclipse
- 3.编译代码：mvn clean compile install
- 4.运行springboot:运行com.bonc.epm.paas.WebAppConfig的main方法或者运行启动命令spring-boot:run
